Origin Story

RepoWeb was built for the Web Technologies course in third-year Computer Science at KNUST, where the capstone is compulsory and everyone gets drafted into build mode. The mission is simple: solve a real-world problem with a real web app, not just a nice slide deck and optimistic promises. Group 43 noticed that schools spend too many human CPU cycles compiling report cards manually, so we built RepoWeb to automate the grind and return valuable time to teachers.
